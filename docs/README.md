## Démarrage rapide

### Je suis visiteur, que puis-je faire ?

Consultez **[USER_COMMERCIAL.md](USER_COMMERCIAL.md)** pour :
- Rechercher une pharmacie
- Trouver une pharmacie de garde
- Consulter les numéros d'urgence
- Proposer une correction ou une nouvelle pharmacie
- Envoyer une suggestion

### Je suis administrateur, comment gérer le site ?

Consultez **[ADMIN_GUIDE.md](ADMIN_GUIDE.md)** pour :
- Accéder au tableau de bord
- Gérer les pharmacies (ajouter, modifier, supprimer)
- Valider les soumissions des utilisateurs
- Configurer le site (SEO, logo, favicon)
- Gérer les publicités et popups
- Consulter les logs d'activité

## À propos de UrgenceGabon.com

**Une plateforme centralisée pour trouver des pharmacies et des numéros d'urgence au Gabon.**

État actuel :
- **89 pharmacies** référencées
- **9 villes** couvertes (Libreville, Port-Gentil, Franceville, Oyem, Mouila, Makokou, Koulamoutou, Moanda, Ntom)
- **Interface mobile-first** responsive sur tous les appareils
- **Système complet d'administration** pour gérer les données
- **SEO optimisé** avec sitemap et robots.txt dynamiques
- **Système de publicités configurable** (images et vidéos)
- **Audit trail complet** de toutes les activités

## Technologie

- **Backend** : Flask (Python)
- **Frontend** : Jinja2 + Tailwind CSS + JavaScript vanilla
- **Base de données** : PostgreSQL
- **Serveur** : Gunicorn
- **Hébergement** : Replit
