# UrgenceGabon - Guide Utilisateur

Bienvenue sur le guide d'utilisation de **UrgenceGabon.com**. Ce document vous explique comment tirer le meilleur parti de la plateforme pour trouver rapidement une pharmacie ou un service d'urgence.

## 1. Trouver une Pharmacie

### Recherche Rapide
1.  Utilisez la **barre de recherche** en haut de la page.
2.  Tapez le nom d'une pharmacie (ex: "Pharmacie du Centre").
3.  Les résultats s'affichent instantanément en dessous.

### Filtrer par Ville
1.  Cliquez sur le nom de votre ville dans la liste des filtres (ex: "Libreville", "Port-Gentil").
2.  Seules les pharmacies de cette ville seront affichées.
3.  Pour voir toutes les villes, cliquez sur "Toutes les villes".

### Pharmacies de Garde (Nuit & Dimanche)
1.  Cliquez sur l'onglet **"Pharmacies de garde"**.
2.  Cette liste affiche uniquement les pharmacies ouvertes actuellement (24h/24).
3.  Vérifiez toujours les horaires indiqués sur la fiche.

## 2. Utiliser la Carte Interactive

L'onglet **"Carte"** vous permet de visualiser les pharmacies autour de vous.

*   **Marqueurs Verts :** Pharmacies ouvertes aux horaires habituels.
*   **Marqueurs Rouges (Clignotants) :** Pharmacies de garde (ouvertes maintenant).
*   **Votre Position (Point Bleu) :** Si vous autorisez la géolocalisation, votre position s'affiche pour vous repérer facilement.

**Astuce :** Cliquez sur un marqueur pour voir le numéro de téléphone et lancer l'itinéraire GPS (Google Maps).

## 3. Numéros d'Urgence

En cas de problème grave, consultez l'onglet **"Urgences"**.

*   **Numéros Nationaux :** Police (177), Pompiers (18), SAMU (1300). Ces numéros fonctionnent partout au Gabon.
*   **Numéros Locaux :** Sélectionnez votre ville pour voir les hôpitaux, cliniques et services d'ambulance spécifiques à votre zone.

## 4. Contribuer à la Communauté

UrgenceGabon est un projet collaboratif. Vous pouvez nous aider à maintenir les informations à jour.

### Signaler une Erreur
Si un numéro de téléphone ou un horaire est incorrect :
1.  Allez sur la fiche de la pharmacie concernée.
2.  Cliquez sur le bouton **"Signaler une erreur"** (icône crayon).
3.  Indiquez la bonne information.
4.  Validez. Un administrateur vérifiera votre modification rapidement.

### Proposer une Localisation GPS
Si une pharmacie n'a pas de position sur la carte :
1.  Cliquez sur **"Proposer coordonnées"** (icône localisation).
2.  Autorisez la géolocalisation ou déplacez le marqueur sur la carte à l'endroit exact.
3.  Envoyez.

### Ajouter une Pharmacie Manquante
Si vous connaissez une pharmacie qui n'est pas listée :
1.  Allez dans l'onglet **"Suggestions"**.
2.  Choisissez "Nouvelle pharmacie".
3.  Remplissez le formulaire avec le maximum d'informations (Nom, Ville, Quartier).
